# Data Analytics Project

## Overview

This project focuses on performing end-to-end data analytics using Python, SQL, and Power BI. The workflow includes data loading, data cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and report generation. The final insights are presented through a Power BI dashboard and a presentation created using Gamma.

The project demonstrates practical skills in data analysis, visualization, database handling, and business reporting.

---

## Dataset

The dataset used in this project contains structured data for analysis and visualization purposes.

### Key Features

* Raw dataset imported using Python
* Cleaned and transformed for analysis
* Stored and queried using SQL databases
* Visualized using Power BI dashboards

## How to Run the Project

### Step 1: Clone the Repository

```bash
git clone <repository-link>
```

### Step 2: Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy
```

### Step 3: Run Python Scripts / Notebook

```bash
jupyter notebook
```

### Step 4: Configure Database

* Create a database in PostgreSQL/MySQL/SQL Server
* Import the cleaned dataset
* Run SQL queries

### Step 5: Open Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop
* Refresh the dataset connection
